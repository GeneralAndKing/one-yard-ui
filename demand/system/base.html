# 系统角色

角色是权限管理中最为重要的一部分！我们内置了一些角色来管理我们的项目，同时他们也各自拥有属于自己的菜单权限，如下：

为了系统的稳定性，我们禁止管理员修改和添加任何权限和角色！

角色名 所属部门 描述 菜单权限
ROLE_ADMIN 系统 管理员 - 仪表盘
- 用户管理
- 权限管理
- 部门管理
- 供应商管理
- 仓库组织管理
- 物料类别管理
- 物料管理
ROLE_PUBLIC 系统 公共 -
ROLE_ACCESS 系统 游客 -

- 需求计划管理
ROLE_PRODUCTION_SUPERVISOR 生产部 生产部门主管 - 当前部门用户管理
- 需求计划管理
ROLE_WORKSHOP_PLANER 车间部 车间部门提报员 - 需求计划创建
- 需求计划管理
ROLE_WORKSHOP_SUPERVISOR 车间部 车间部门主管 - 当前部门用户管理
